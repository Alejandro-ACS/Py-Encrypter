# Py-Encrypter
Encrypts all messages using the pi number.
# Requirements
You need to install the json and requests libraries with the nexts command.
```bash
pip install random2
```
# Run file
Just download the Encrypter.py file and execute the following command in the console.
```bash
python Encrypter.py
```
